Generative Engine Optimization toolkit for WordPress. Auto-generates llms.txt and llms-full.txt, audits AI crawler access.

== Description ==

**Big GEO** is a comprehensive Generative Engine Optimization (GEO) plugin that helps your WordPress site become discoverable by AI engines like ChatGPT, Claude, Perplexity, and Google Gemini.

= Features =

* **llms.txt Generator** - Automatically creates a discoverable llms.txt file with all your post titles and URLs in Markdown format
* **llms-full.txt Generator** - Optional full-content export for AI training and indexing
* **AI Crawler Audit** - Detects and reports which AI bots can crawl your site via robots.txt
* **3-Tier Robots.txt Management** - Handles virtual WordPress robots.txt, physical files, and custom URLs
* **Auto-Regeneration** - Updates llms.txt whenever you publish or update posts
* **SEO Plugin Integration** - Respects noindex settings from Yoast, Rank Math, and All in One SEO
* **Dashboard Widget** - Quick status overview in WordPress admin

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/knr-geo/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Big GEO screen to configure

== Frequently Asked Questions ==

= What is llms.txt? =

llms.txt is a proposed standard for making your site's content discoverable by Large Language Models (LLMs) and AI search engines.

= Which AI bots are supported? =

Big GEO audits: GPTBot (ChatGPT), ClaudeBot, PerplexityBot, Google-Extended (Gemini), Amazonbot, and cohere-ai.
